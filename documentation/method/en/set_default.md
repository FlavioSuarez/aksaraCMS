### Reference
`set_default($foo, $bar)`

**Parameter**
* **$foo** [`string`] *the detail related to the variable;*
* **$bar** [`string`] *the detail related to the variable.*

&nbsp;

### Usage Sample
`$this->set_default('foo', 'bar');`

`$this->set_default('baz', 'qux');`

**You can use this method in groups as below:**
```php
$this->set_default([
    'foo' => 'bar',
    'baz' => 'qux'
]);
```

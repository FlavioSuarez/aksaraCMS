### Usage Sample
`$this->insert_id('foo', 'bar');`

`$this->insert_id('baz', 'qux');`

**You can use this method in groups as below:**
```php
$this->insert_id([
    'foo' => 'bar',
    'baz' => 'qux'
]);
```

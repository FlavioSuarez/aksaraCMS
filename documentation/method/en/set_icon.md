### Usage Sample
`$this->set_icon('foo', 'bar');`

`$this->set_icon('baz', 'qux');`

**You can use this method in groups as below:**
```php
$this->set_icon([
    'foo' => 'bar',
    'baz' => 'qux'
]);
```
